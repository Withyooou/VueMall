# vuemall

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```



## About

本项目是基于coderwhy老师的Vue教学视频而完成的，在原项目的基础上做了一定程度的提升，非常适合Vue初学者练习，特此感谢！
